# Lab3-machine-learning-EDA
Heart Disease EDA – Lab Assignment

This lab focuses on performing Exploratory Data Analysis (EDA) on a heart disease dataset to understand its structure and identify important patterns.

Data Preparation:

-Converted incorrectly formatted columns to numeric types

-Handled missing values

-Removed columns with excessive missing data


Analysis Performed:

-Distribution plots

-Scatter plots

-Correlation heatmap

-Group-based analysis
